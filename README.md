# lgoritmos-logica-c
Coleção de algoritmos desenvolvidos como atividades acadêmicas durante o curso de Análise e Desenvolvimento de Sistemas. O foco destes projetos foi aplicar conceitos fundamentais de programação estruturada.
Conceitos e Estruturas Aplicadas
Nestas atividades, utilizei:

Estruturas de Decisão: if, else if e else para validações e classificações (ex: IMC e Segurança) .

Seleção Múltipla: switch-case para menus interativos (ex: Conversor de Temperaturas e Peso nos Planetas) .

Laços de Repetição: do-while e while para manter os programas em execução até o usuário decidir sair.

Entrada e Saída: Manipulação de dados com printf e scanf .

Modularização: Criação de funções específicas para cálculos e verificações .
